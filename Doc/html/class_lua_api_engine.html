Detailed Description

LUA API functions.

Stores all the definitions of API functions available to users. Functions with pattern "prepareX" are the preparing functions that are invoked during lua's engine parsing. The functions put the commands (pointers to functions without 'prepare' prefix) to the LuaEngine object. See example for LuaEngine.

Field that maps string to Key object.
