Member Function Documentation

void CalendarTools::disableForAction ( )
static

This method is called when an action with repetitions has been selected on the calendar to enable buttons for operating on actions.

This method will be called when there are more than once actions on actions list.

void CalendarTools::enableForAction ( Action *  action)
static

This method is called when an action without repetitions has been selected on the calendar to enable buttons for operating on actions.

This method will be called when there is only one action on actions list.
