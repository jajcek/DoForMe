void TraySystem::addQAction ( QAction *  action)

Adds operation for the context menu (not the actions from the calendar), e.g. Open or Exit.

Beware that it is necessary to modify the method, because it doesn't use any loop to obtain all operations. It was needed to put actions from calendar between some operations that's why the programmer sets the order.
