\subsection{Detailed Description}
Singleton used for calling actions from the calendar. 

\hyperlink{class_action_caller_ab62a3a145a4bf356301223626c971caa}{Action\-Caller\-::get\-Instance} returns the only object. By using \hyperlink{class_action_caller_ae05e223ea1a75f49b694654a2e3267a2}{Action\-Caller\-::set\-Actions} we are able to set actions that will be called by the \hyperlink{class_action_caller}{Action\-Caller}'s timer. The timer is set when settings action in \hyperlink{class_action_caller}{Action\-Caller}\-:set\-Action method. \begin{DoxyAuthor}{Author}
